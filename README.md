Esse curso eu estudo a criação e contribuição com projetos open source.

Suportada versão 3 de Python

Para instalar:

```console
Python3 -r venv .venv
source venv\bin\activate
pip install -r requiments-dev.txt
```

Para conferir qualidade de código:

```console
flake8
```
